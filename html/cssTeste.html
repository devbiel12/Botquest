Teste seu conhecimento em CSS3

Tendo lido o conteudo e entendido ele, agora você irá testar seu conhecimento nessa linguagem. Abaixo haverão 10 perguntas sobre o texto de CSS3 com 3 alternativas cada e somente uma correta. Boa sorte!

Teste de CSS3

1. O que é CSS?

Uma linguagem de programação para criar páginas da web.
Uma linguagem de estilo para adicionar estilos visuais às páginas da web.
 Uma linguagem para criar aplicativos móveis.

2. O que as regras CSS consistem?

Uma série de declarações.
Um seletor e um conjunto de declarações.
Um conjunto de propriedades.

3. O que é um seletor em CSS?

Um estilo de fonte.
Uma propriedade visual.
Um identificador que especifica qual elemento HTML a regra deve ser aplicada.

4. Como o CSS pode ser vinculado a uma página HTML?

Usando uma tag script.
Usando uma tag style.
Usando uma tag link.

5. O que as propriedades CSS podem ser usadas para criar?

Animações e transições.
Arquivos de imagem.
Funções matemáticas.

6. O que o CSS permite que os desenvolvedores criem para tornar o layout responsivo?

Imagens interativas.
Animações complexas.
Layouts que se adaptam a diferentes dispositivos e tamanhos de tela.

7. Qual é a vantagem de usar CSS para estilizar páginas da web?

Ele permite que os desenvolvedores criem layouts complexos.
Ele separa o conteúdo da página de sua apresentação visual.
Ele adiciona interatividade à página.

8. Qual é o objetivo principal do CSS?

Fornecer uma linguagem para criar aplicativos web.
Adicionar funcionalidade aos elementos HTML.
Estilizar elementos HTML e controlar a aparência da página.

9. Qual é o benefício de escrever CSS em um arquivo separado?

É mais fácil para os desenvolvedores iniciantes aprenderem.
Torna mais fácil para os desenvolvedores manter e modificar a aparência da página.
Faz com que a página carregue mais lentamente.

10. O que é uma tag ?

Uma tag HTML usada para criar links para outras páginas.
Uma tag HTML usada para incluir arquivos de script na página.
Uma tag HTML usada para vincular um arquivo CSS à página.
