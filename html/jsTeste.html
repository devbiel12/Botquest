Teste seu conhecimento em JS

Tendo lido o conteudo e entendido ele, agora você irá testar seu conhecimento nessa linguagem. Abaixo haverão 10 perguntas sobre o texto de JS com 3 alternativas cada e somente uma correta. Boa sorte!

Teste de JS

1. Qual é a principal característica do JavaScript?

Capacidade de manipular o HTML e o CSS em tempo real
Ser executado apenas no lado do servidor
 Criar aplicativos móveis nativos

2. O que é possível desenvolver usando JavaScript?

Efeitos visuais e animações
Banco de dados relacionais
Sistemas operacionais

3. O que significa o fato de o JavaScript ser uma linguagem baseada em eventos?

Ele pode responder a ações do usuário, como clique de botões e rolagem da página
Ele só pode ser executado em plataformas específicas
Ele é uma linguagem de programação baseada em hardware

4. Em que plataformas o JavaScript é executado?

Desktops, dispositivos móveis e navegadores
Apenas em desktops
Apenas em dispositivos móveis

5. O que é possível comunicar usando JavaScript?

Servidores, permitindo a atualização de informações sem a necessidade de recarregar a página inteira
Apenas com outros scripts JavaScript
Apenas com servidores locais

6. O que é o Node.js?

Um ambiente de tempo de execução JavaScript do lado do servidor
Um editor de texto para JavaScript
Um framework para desenvolvimento de jogos em JavaScript

7. O que é uma biblioteca em JavaScript?

Um conjunto de funções pré-escritas para facilitar o desenvolvimento
Um arquivo de configuração de servidor
Um conjunto de modelos para criar páginas web

8. Qual é a importância da comunidade de JavaScript?

Contribui com bibliotecas e frameworks para tornar o desenvolvimento mais fácil e eficiente
Limita o desenvolvimento de novas tecnologias
Torna o JavaScript obsoleto em relação a outras linguagens de programação

9. Qual é a diferença entre JavaScript e outras linguagens de programação?

JavaScript é executado no lado do cliente, enquanto outras linguagens podem ser executadas tanto no lado do cliente quanto no lado do servidor
JavaScript é a única linguagem que permite manipular HTML e CSS em tempo real
JavaScript não suporta programação orientada a objetos

10. Qual é a importância do JavaScript para o desenvolvimento web? ?

É essencial para a criação de páginas web interativas e aplicativos web
É apenas uma linguagem secundária usada para pequenos efeitos visuais
É obsoleto e não é mais usado pelos desenvolvedores web.
